## üîß Usage

### Getting Started
1. Open `index.html` in your web browser
2. Fill out the letter form with recipient and content details
3. Click "Preview Letter" to see the formatted result
4. Use zoom controls to review the letter
5. Click "Generate PDF" to download your professional letter

### Form Fields
- **Recipient Name**: Required - Person or organization receiving the letter
- **Title/Position**: Optional - Recipient's job title or position
- **Recipient Address**: Required - Complete mailing address
- **Date**: Auto-filled with today's date, can be modified
- **Reference Number**: Optional - Internal reference or file number
- **Subject**: Required - Brief description of letter purpose
- **Salutation**: Dropdown with professional greetings
- **Letter Body**: Required - Main content of the letter
- **Closing**: Dropdown with professional closings
- **Signatory Name**: Optional - Name of person signing the letter

### Best Practices
- **Keep it Professional**: Use formal language and proper business format
- **Be Concise**: Clear, direct communication works best
- **Proofread**: Review the preview carefully before generating PDF
- **Save Drafts**: The system auto-saves, but you can manually save important drafts
